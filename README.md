# 🛡️ Automated Security Audit Tool

A Python tool that automatically scans IT infrastructure 
for vulnerabilities, weak configurations, and compliance 
risks — generating detailed security reports.

## 🔍 Features

- Port scanning & open service detection
- Weak password configuration checks
- SSL/TLS certificate validation
- Outdated software version detection
- Automated PDF security report generation

## 🧰 Tech Stack

Python · Socket · SSL · Subprocess · JSON · Logging

## 🚀 How to Run

```bash
git clone https://github.com/Nwajei-Ndubuisi-Francis/automated-security-audit-tool.git
cd automated-security-audit-tool
pip install -r requirements.txt
python audit.py --target 192.168.1.1
